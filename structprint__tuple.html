Detailed Description

template<size_t cur_index, size_t tuple_size, typename... Args>
struct print_tuple< cur_index, tuple_size, Args >

Метафункция, которая разворачивается в кучу классов

Её задача определить, что все типы кортежа одинаковы. Работает поверх std::is_same_v<...>

Parameters
cur_index- номер элемента в кортеже, начинает с нуля, затем увеличивается компилятором на каждом этапе генерации очередного класса, пока не опустеет кортеж
tuple_size- сколько элементвов в кортеже, определяется при первом вождении и тащится через все нагенерированные классы, пока cur_index не совпадет с tuple_size
Args- весь кортеж, как он есть, пробрасывается через все сгенерированные реализации. В каждой реализации извлекается нуждый элемент по увеличивающемуся cur_index
